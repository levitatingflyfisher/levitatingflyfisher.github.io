What this is — and isn't

  • Is: a read-only dashboard for journals you already have. It never changes a byte of your data.
  • Isn't: the full app. Importing statements, categorizing, budgeting and editing all live in the glean desktop app (Windows / macOS / Linux).
  • Private by construction: there is no server. The math runs in a WebAssembly sandbox on this device; close the tab and nothing remains.
